# LLM-Based-NPC

Repository to validate the effectiveness of RAG and memory management techniques for LLMs, built using LangGraph.
Where the ultimate aim of the repository was to evaluate how effective LLMs were at imitating characters from the game Elden Ring, utilizing a variety of memory trimming and management techniques.
This involved testing whether the LLMs remembered facts about the user throughout chat sessions, and relevant facts about the Game World that said character should know.

Inside the files are some directions for how to run this repository.
